Stuctured Query Language

SQL or Structured Query Language is the primary language used within SSMS (SQL Server Management Studio). Using SQL takes getting used to, but is considered one of the easier languages to learn due to it's similarity with the english language.
In other words, the keywords one would use include SELECT, FROM, WHERE, and CREATE.
A basic SQL query would look like this: SELECT EmployeeID, EmployeeName FROM Employee
Which would return a small table like this:

SQL Query Results
EmployeeID
 EmployeeName

 2  Joe
 3  Jared
 4  Jack

Overall, SQL can be quite useful for obtaining information in a database, and it is definitely a valuable language to learn.
