What Exactly is A Table?

A table is an incredibly useful tool used to organize and display data in a way that makes sense. A table consists of rows and columns. The columns are usually the header for the information displayed at the level below it, the rows. Rows will show data related to the header. For example, the column "video format" indicated the data below (the rows) will show video formats such as .wav or .mov. Tables can be created in HTML by using the <table> tag. Furthermore, you can add the column names by using the <th>, rows with <tr>, and data with <td>.
More examples can be see at this link: w3schools.

Database Terms
Term Definition
Primary Key A field that uniquely identifies all the tables records Ex: FlightID in Flights table
Foreign Key A field that uniquely identifies another row in a different table using the first tables primary key Ex: FlightID in Customer table
Structured Query Language (SQL) Database language used interact with the database management system. Ex: SELECT * FROM Customer
Database Management System (DBMS) The software involved that stores, retrives, and updates data on a database server. Microsoft SQL Server, Oracle, etc
