Video and Audio Tags

For the chapter 8 video, I created a PowerPoint presentation and exported it to a 1080p video. The default file type for the video when exported was MP4 which I then converted using the html tags to OGG. From there, I added some attributes to the video tag such as width, height, poster, controls and the error message if no video is displayed for some reason. The width and height took some playing around with to make it fit the page right, but it did not take too long to figure it out. The poster image I used was snipped from the end of the presentation. I did not use any extra software in the process, as the html converted it on the web page (unless I'm missing something).
The chapter 9 audio I recorded using the Windows 10 application, Voice Recorder which saved as M4A. Instead of using video tags, I used audio tags. I enabled the controls and added an error message. The audio is being converted to two different file types in the html, One as OGG, and the other as mpeg.
