Various Protocols

Popular Protocols in Information Technology

PPP or Point to Point Protocol is a computer network protocol to transport multiprotocol data over point-to-point links.

PPoE or Point to Point over Ethernet uses PPP, and runs over broadband connections, using an ethernet link layer protocol allowing the user to connect to hosts over bridges and concentrators.

HTTP or Hypertext Transfer Protocol allows computers all over the world to communicate via the World Wide Web. It provides a definition for how pages are formatted and transmitted and how servers and browsers handles various responses and commands.

FTP or File Transfer Protocol is what is primarily used to transfer files over the web. FTP works off the TCP/IP protocols to achieve this.

SMTP or Simple Mail Transfer Protocol uses the TCP/IP protocol to send and receive emails. It is somewhat limited in receiving mail, so it is usually used as a sending protocol and one of the other two, POP3, or IMAP is used as receiving instead.

IMAP or Internet Message Access Protocol is another standard email protocol. It allows the user to perform their everyday actions from a web server. Generally, IMAP is thought as a remote file server while POP3 is the opposite.

POP3 or Post Office Protocol is an email protocol used to retrieve email from a remove server over TCP/IP connection. POP3 is starting to lose its popularity to IMAP due to its more advanced features.

NNTP or Network News Transfer Protocol is used by computers and servers for managing the notes posted on UseNet newgroups. NNTP servers run the global network of collected Usenet newgroups and include the server at your internet access provider.
