The Javascript I included on this page is very basic, but it demonstrates one use of how Javascript can be used. In the example above, clicking the button changes the text "this is a demonstration" to "Hello JavaScript!". They do this by linking the button click to a Javascript function. When the function is called, the text is displayed.

This example is also known as Client-side Javascript or CSJS. Client side Javascript enhances and changes web pages. This is quite different from Server-side Javascript or SSJS where the Javascript would allow one to access databases, servers, etc. They both use Javascript as the primary language; however, their purposes are different.

In Javascript, objects are organized into a model known as the Document Object Model or DOM. The DOM is structured as a tree where the document itself is at the top of the tree. According to the W3C, the DOM allows programs and scripts to dynamically access and update content, structure, or style of a document. In other words, this concept of being able to update any page via Javascript or any other language is known as Dynamic HTML or DHTML.

Application Programming interface or API are very popular in the world of web development. APIs are applications that allow other programs to communicate with each other. As an example, Google's APIs allow web developers to implement Google apps such as Calendar, Maps, and Youtube into their websites. There are tons of other APIs as well from popular websites such as Facebook, Yahoo, Twitter, etc.
